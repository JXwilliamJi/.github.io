如何选购 SSD

发表于 2020-09-02 | 分类于 生活 |
字数统计: 1.7k | 阅读时长 ≈ 6

SSD是什么

SSD 即Solid State Disk首字母缩写，翻译为固态硬盘。

SSD寿命

读写寿命

为什么SSD有寿命？ SSD通过对浮置栅极的充放电识别0和1。每次加电压才能充电，加电压会损耗绝缘层。所以SSD的寿命取决于绝缘层的厚度和充电次数，NAND擦写需要损耗浮置栅极上的绝缘层，绝缘层只要漏电，SSD就无法存储数据，即损坏，所以SSD寿命约等于所有cell擦写寿命。

与SSD的NAND颗粒相比，内存条的DRAM颗粒没有读写寿命，理论是无限次读写，但是他是易失性存储，意味着不通电数据就会丢失。

制程制约

通常制程越先进，寿命反而越短。

  • 工艺越先进 –> 晶体管越小 –> 绝缘层越窄 –> 擦除越容易磨损绝缘层 –> 电子越容易逃逸 –> 寿命越短

  • 工艺越先进 –> 晶体管越小 –> 浮栅中能容纳的电子越少 –> 电子逃逸后电压变化大 –> 寿命越短

SSD 容量

厂商标注的500G为何检测就剩下470G？

厂商出厂是按照1KB = 1000B，1MB = 1000KB换算，1GB = 10⁹B。

而电脑是二进制的，1KB在电脑上等于1024B。1GB = 2³⁰B。

因此厂商标注的1GB电脑会显示0.9313G，如500G显示约为465.66 G。

SSD 的组成

SSD 通常由主控芯片、缓存颗粒、闪存芯片组成。

主控芯片

主控芯片就是 SSD 的大脑，它涉及到数据处理能力、算法、读写等操作。好的主控在性能和传输速度会比一般的主控快数倍以上。

缓存颗粒

缓存颗粒主要用于小文件的缓存，防止对闪存芯片过多的存取，有效提高SSD寿命。有些廉价的SSD为了节省成本会省去这部分，导致寿命缩减。目前TLC、QLC通过做大缓存来加快读取速度，让用户感觉不出速度的差异，但是一旦缓存满，读写速度会骤降。

闪存芯片

闪存芯片分为NOR Flash（或非）和 NAND Flash（与非）闪存芯片。固态常见的是NAND闪存芯片。目前闪存因为存储层叠的不同分为SLC（单层单元）、MLC（多层单元）、TLC（三层单元）、QLC（四层单元）、研制中的PLC（五层单元）。

SSD 层叠结构

先上一张正确的存储示意图

SLC（单阶单元）Single-Level Cell

1bit/cell，代表每个单元存储1bit的数据。约有10万次的擦写寿命（P/E寿命），但是容量小，价格贵。

MLC（双阶单元） Multi-Level Cell

2bit/cell，速度一般寿命一般，价格一般，约3000—1万次擦写寿命。

TLC（三阶单元） Trinary-Level Cell

3bit/cell，也有Flash厂家叫8LC，速度慢寿命最短，价格便宜，约500-1千次擦写寿命。

QLC（四阶单元）

4bits/cell,成本更低，容量更大，但寿命更短（理论可擦写150次）。

PLC (五阶单元)

5bits/cell,成本更低，容量更大，但寿命更短（理论可擦写35次）。

可擦写次数看看就好，实际意义不大，未来随着技术的进步加厚浮置栅极上的绝缘层或者替换更耐久的绝缘层就会有根本性改变

SSD 选购关注的重要参数

制程

制程越先进，寿命越短（具体原因是漏电）。相比较19nm的我们尽量选择40nm的TLC。

TBW

TBW意味着保修范围内的总的可写入数据量,通常为300TBW，高端可达600TBW。这个才是关键参数，我们一般用户购买SSD只需要关注这个参数就可以了。SSD的发展归根结底是TBW的发展和每GB成本的降低。

300TBW 意味着总共写入的数据超过300TB（300，000GB）就过保修。

P/E擦写次数

得益于动态平均抹写技术，主控芯片会将数据均匀的写入每个cell中，当写入的数据量达到SSD的最大容量，意味着已经对每一个cell进行过一次擦写，记为一次P/E。单个cell的擦写次数毫无意义。

P/E擦写次数和TBW有换算关系， P/E = TBW / 硬盘容量。例如标注300TBW，硬盘容量为500GB（0.5TB），那么 P/E次数 = 300 / 0.5 = 600次（注意单位）。

但是并不是600次P/E就会坏，达到600次P/E意味着过保。一般SSD会达到1.5倍的P/E擦写次数。

大的缓存可以减少不必要的擦写，延长SSD寿命。

优质的原厂TLC闪存配合具备LDPC纠错的主控芯片，可实现3000次左右的擦写寿命，接近MLC的寿命。

缓存

缓存不仅可以避免无意义的擦写，还在于提高读写速度。缓存越大，小于缓存的文件读写都能保持高速读写，一旦超过缓存，速度会骤降，所以大缓存很重要。

容量

利用3D NAND技术，即垂直堆叠多层数据存储单元，SSD打破了2D NAND物理数据存储极限。目前最高堆叠高度达到128层，单Die最大可以达到512Gb，通过BGA封装，一个黑色闪存芯片可以封装2-4个Die。

未来如果达到1000层甚至更多，SSD容量也会越来越大，越来越便宜。

SSD 常见问题

  • 测速软件有用么

有点用，测速软件检测的体验速度。很多人认为SLC Cache是欺骗技术，那你电脑里插个内存其实也是欺骗技术。SLC Cache确实可以加快用户端的体验速度，SSD里面如何优化的用户并不关心。但是SSD一旦存储超过90%数据，快速写入会失效。

  • SSD TLC速度真的慢么

因为SLC Cache的存在，用户其实感知不到差异，但是一旦Cache满了，速度就会骤降。TLC理论速度在250MB/S。

  • 三星SSD MLC比TLC好么

MLC优点在于速度，MLC的TBW也比较高，其他差距不大。

  • 为什么有些杂牌SSD如此便宜

NAND颗粒分为原片、黑片、白片。SSD有各种牌子，但是NAND颗粒就5家。每一家检测合格并且贴牌的为原片，质量优；白片就是不明厂商封装的颗粒，质量未知，赌人品；黑片就是原厂检测中不合格而淘汰的下脚料。杂牌SSD一般使用后两者以节省成本。

SSD 未来趋势

3D XPoint 技术提供了比较NAND 10倍的闪存性能，和1000倍的耐久性。为未来的存储提供了更大的可能性，目前瓶颈在于成本。

结论

重点了，大家购买时请选择好品牌，大缓存，高TBW，500G以上的SSD。
