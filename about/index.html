自我介绍下,本人江苏泰兴人,北京工作,可以称呼我为 Ji.

Ji 从事的是 iOS 开发工作,从事的金融私募基金相关的开发工作.

Ji 的技术一般,远远无法和iOS 圈内的大牛相提并论,写博客的目的仅仅是记录自己的学习轨迹,如果无意中能够为朋友们提供微不足道的帮助,那是我的荣幸,如果有错误,请及时留言,我会及时修改博客.

Ji 主要关注的点是音视频相关,图形学,iOS 逆向相关.其实这两块非常大,学习过程中会发现涉及到的知识也是非常广泛. 所以 Ji 目前直接通过研究 Unity3D 来节省时间,方便学习.

Ji 学习了很多语言, iOS 相关的有 OC,Swift, iOS 基础相关的有 C++, 目前正在学习 C++, 为了打包和组件化,使用fastlane学习了 Ruby, 为了学习 Unity3D 学习了 C#, 为了移动端的 web 开发学写了前端 HTML/CSS/JS, 同时还学习了 Python,未来还打算学些 Java.

其实学习了这么多语言,总结下来,会发现, Ruby Python 通过对齐来组织语言, JS 和 Swift 语言相似, C++ 和 C# 基本相同,所有语言需要熟悉的就是 API 以及 对应的语法糖.

同时研究逆向还关注了汇编, 会一点8086和 ARM 汇编,能够勉强看懂. 主要是为了验证 xx 比 xx 效率更高的言论.

Ji 在目前的公司已经呆了快3年了,很多朋友都跳槽,薪资蹭蹭蹭的涨, Ji 心里其实也希望薪资的增长.但是在目前的公司,有充分的时间学习,友好的同事关系 和 新技术的不断使用让 Ji 决定安心待着.

Ji 希望能够结交各行各业的朋友,不需要是大牛,只要能聊得来就好.当然需要的是平等的输入输出,如果只是为了让 Ji 不断输出, Ji 也是不干的…

Ji 目前已经通过了信息系统项目管理师,未来打算继续学习 PMP ,了解管理知识.也能从管理的角度来了解领导,也能提升自己.

好了,就扯这么多…
